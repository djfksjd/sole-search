# 지역 소스 레지스트리

프로필의 `province`에 해당하는 행만 사용한다. 상태는 전부 `manual`(초기 릴리스) —
보고서에 해당 지역 재단·기관의 확인 절차를 첨부하고 coverage_manifest에 manual로 기록한다.
URL이 `koreg 경유`인 지역은 신용보증재단중앙회에서 해당 재단 링크를 확인한다 (추정 URL 금지).
